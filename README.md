# intership

Local senior internship finder built from the Interlake internship/job fair catalogs.

Open the site locally from:

```text
internship_research/senior_internship_finder.html
```

Main files:

- `internship_research/senior_internship_finder.html` - standalone searchable website.
- `internship_research/internship_profiles_copyable.md` - copy/paste profile catalog.
- `internship_research/internship_entries.csv` - raw placement rows.
- `internship_research/senior_internship_finder.data.json` - website data.
- `internship_research/build_internship_catalog.py` and `build_deliverables.py` - rebuild scripts.
